# 2026年全国大学生嵌入式芯片与系统设计竞赛·ST赛道

​																							By:ecat

## 作品简介：

​	文件泄密，数据丢失，存储设备损坏，云盘安全性较低，是我们日常工作中经常遇到的问题，针对此一难题，我们团队基于STM32H5为核心，充分利用STM32H563的TrustZone 安全架构、硬件加密加速器和安全启动机制，开发出了一款非常规加密U盘 + NAS协同存储系统。作品创新提出“隐式触发 + 分级认证”安全机制，仅当用户输入自定义非常规键位组合时设备才被枚举识别，从交互层隐藏设备存在；随后通过口令进入加密空间，实现写入即加密。NAS 端采用 MD5 摘要认证与权限隔离，实现多终端安全共享与集中管理。系统兼顾安全性、隐蔽性与扩展性，适用于隐私保护、团队协作及移动安全存储场景，具有较强工程与应用价值。
